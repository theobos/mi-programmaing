MATRMD Selection Values

01 - Processor Utilization
03 - Storage Management Counters
04 - Storage Transient Pool Information
08 - Machine Address Treshold Data
09 - Main Storage Pool Information
0A - MPL Control Data with 2-byte counts
0C - Machine Reserved Storage Pool Information
11 - User Storage Area 1 - OBSOLETE
12 - Auxiliary Storage Information
13 - Multiprocessor Utilizations
14 - Storage Pool Tuning
15 - Delay Cost Scheduling Information
16 - MPL Control Information
17 - Allocation And De-allocation Counts Per Task And Thread
18 - Processor Multi-tasking Mode
19 - Dynamic Priority Adjustment Mode
1A - Disk Collection/balancing Status
1B - Mapping Of Partition Processors
1C - DASD Management Status
1D - Disk Information
1E - Interactive Utilization Data
1F - Auxiliary Storage Pool Information (short format)
20 - Auxiliary Storage Information Including Varied-off Independent ASPs
