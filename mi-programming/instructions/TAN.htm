Tangent (TAN)

Bound program access
Built-in number for TAN is 402.
TAN (
     source   : floating point(8) value
) :   floating point(8) value which is the 
      tangent of the source value

Description

The tangent of the numeric value of the source operand, whose value is considered to be in radians, is computed and the result is returned.

The result is in the range: 

-infinity <= TAN(source) <= +infinity

See floating point results from special values for additional information.
