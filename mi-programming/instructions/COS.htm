Cosine (COS)

Bound program access 


Built-in number for COS is 400.
COS (
     source   : floating point(8) value
) :   floating point(8) value which is the cosine of the 
      source value
Description

The cosine of the numeric value of the source operand, whose value is considered to be in radians, is computed and the result is returned.

The result is in the range: 

-1 <= COS(source) <= 1

See floating point results from special values for additional information.
