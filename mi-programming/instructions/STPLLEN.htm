Store Parameter List Length (STPLLEN)


Op Code (Hex) Operand 1


0241 Length


Operand 1: Binary variable scalar.

Description

A value is returned in operand 1 that represents the number of parameters associated with the invocation's external entry point for which arguments have been passed on the preceding Call External (CALLX) or Transfer Control (XCTL) instruction.

The value can range from 0 (no parameters were received) to the maximum size possible for the parameter list associated with the external entry point.
