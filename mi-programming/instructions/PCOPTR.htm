PCO Pointer (PCOPTR)

Bound program access
Built-in number for PCOPTR is 144.
PCOPTR (
) :   space pointer(16) to the PCO associated with the 
      activation

Description

A space pointer is returned, which points to the first byte of the PCO (process communication object) for the process which owns the program activation associated with the current invocation. If the current invocation is a client executing in a shared activation group then the pointer returned refers to an object owned by a process other than the current process.
