Return PCO Pointer (PCOPTR2)

Bound program access
Built-in number for PCOPTR2 is 358.
PCOPTR2 (
) :   space pointer(16) to the process' PCO 
      (process communication object)

Description

Return PCO Pointer (PCOPTR2) obtains addressability to a process' PCO (process communication object) and returns it in a space pointer.

This built-in function supports a high performance alternative method of obtaining the PCO pointer of the currently executing process.
Note: Another alternative for obtaining the PCO pointer is via option hex 17 on the Materialize Process Attributes (MATPRATR) instruction.
