Compute Length of Null-Terminated String (STRLENNULL)

Bound program access
Built-in number for STRLENNULL is 23.
STRLENNULL (
            null_terminated_string   : address of aggregate(*)
) :   unsigned binary(4) value which specifies the number of 
      bytes between the beginning of the string and the first
      null (ie. zero) byte

Description

The null terminated string is searched for the first null byte (hex 00). The number of non-null bytes found is returned.
