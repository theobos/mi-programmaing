Retrieve Teraspace Address From Space Pointer (RETTSADR)

Bound program access
Built-in number for RETTSADR is 623.
RETTSADR (
          source_pointer   : space pointer(16)
) :   local form address

Description

Retrieve the teraspace address from source pointer. If source pointer contains a null pointer value then a null pointer value is returned.

If source pointer contains a space address that does not point to teraspace, then a space address is not a teraspace storage address  (hex 0609) exception is signalled. If any exception is signalled during this operation the result is undefined.

This instruction may be specified in a program template only when the (bound) program is to be created as teraspace capable.

Lock Enforcement
