2C15

Invalid Operation for Program

The operation requested is not supported for this type of program. The system supports four types of program objects:

  • Non-bound program
  • Bound program
  • Bound service program
  • Java program

Not all operations are supported for all type of programs. (E.g., a bound service program object cannot be deactivated.)

  • Activate Program (ACTPG)
  • Call External (CALLX)
  • Deactivate Program (DEACTPG)
  • Materialize Activation Export (MATACTEX)
  • Reinitialize Static Storage (RINZSTAT)
  • Transfer Control (XCTL)
