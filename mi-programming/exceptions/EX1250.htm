1250

Referential Constraint Violation

An attempt has been made to change a data space entry such that a foreign or parent key constraint would be violated. For operations which delete a data space entry, a parent key would be deleted which had a corresponding equal key in a foreign key index. For operations which insert a data space entry, a foreign key would be inserted which did not have an equivalent parent key. For update operations, a parent key value is changed that has a corresponding foreign key value or an updated foreign key value does not have a matching parent key.
