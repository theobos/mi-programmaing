<P><B><U>Ordinal entry number</U></B> will contain a value of binary 0
when the exception occurs when inserting new data space entries or
performing group-by and join operations.
<P><B><U>Number of fields in error</U></B> contains the count of fields
with errors.
<P><B><U>Field data</U></B>
<UL>
<P><LI><B><U>Field number</U></B> is the relative location of the field as
specified when creating the cursor.
A <I>field number</I> of 1 is the first field in the data field
location.
<P><LI><B><U>Error type</U></B> values are as follows:
<TABLE CELLPADDING="3">
<TR VALIGN="TOP"><TD><P>Hex 0001 = </TD><TD><P>Conversion:
An invalid character was found by the derived operation, Invalid
Character Scan.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 0002 = </TD><TD><P>Decimal Data:
(1) Sign encoding is invalid for packed or zoned format, or (2) digit
encoding is invalid for packed or zoned format.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 0006 = </TD><TD><P>Floating-Point Overflow:
During conversion, a floating-point value exceeded the maximum value
that can be represented.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 0007 = </TD><TD><P>Floating-Point Underflow:
During conversion, a floating-point value became less than the minimum
value that can be represented.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 0008 = </TD><TD><P>Length Conformance:
A length specified for a variable length field was outside the limits of
that field.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 0009 = </TD><TD><P>Floating-Point Invalid Operand:
A floating-point NaN was used as an operand to convert from long
floating-point format to short floating-point format or from short
floating-point format to long floating-point format.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 000A = </TD><TD><P>Size:
The destination field is too small to hold all significant digits of the
source field.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 000C = </TD><TD><P>Invalid Floating-Point Conversion:
An invalid floating-point value was used as an operand to convert
floating-point to packed decimal or floating-point to binary.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 000D = </TD><TD><P>Floating-Point Inexact Result:
In a conversion operation, a floating-point value had at least one bit
of precision rounded away.
</TD></TR><TR VALIGN="TOP"><TD><P>Hex 0012 = </TD><TD><P> Invalid extended character data:

</TD></TR><TR VALIGN="TOP"><TD><P>Hex 0015 = </TD><TD><P>Date boundary overflow:
