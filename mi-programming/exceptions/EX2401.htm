2401

Pointer Does Not Exist

A pointer reference was made to a storage location in a space that does not contain a pointer data object, or a reference was made to a space pointer machine object that was not set to address a space.

This exception may not be signaled for operations which refer to the value of a pointer, but which do not attempt to reference the data or object the pointer addresses.

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • Any instruction that has pointer operands
