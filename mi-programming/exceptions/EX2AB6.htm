An entry in the procedure order list was not valid.

Pointer to the procedure order list is a space pointer to the input procedure order list referential extension.

Byte offset indicates the offset of the field within the procedure order list referential extension that is invalid.

Entry index indicates which entry after the header in the procedure order list referential extension is invalid.

Field length indicates the size of the invalid field within the procedure order list referential extension.

Bit offset indicates the offset of the bit that is invalid within the procedure order list referential extension.

Reason code identifies the reason for the failure.
