4805

MI Wait in Kernel Mode Terminated

A condition occurred while in MI wait when the thread execution mode is set to kernel mode which caused the MI wait to be terminated. The exception is detected after the MI wait has been entered. The cancel reason has meaning only when the termination code is set to invocation cancelled (hex 0007), otherwise it will be set to binary 0.

The execution of the instruction is suppressed when the exception is signaled.
