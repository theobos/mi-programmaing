4401

Object Domain or Hardware Storage Protection Violation

An attempt was made to use a blocked instruction or to access protected storage. Protected storage is in the system domain or has storage protection attributes that restrict its use. Only programs running with system state can use blocked instructions.

The execution of the instruction is suppressed when the exception is signalled.

Note: Below, IPS stands for "implicit process space".

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • Any instruction that contains an operand that can be a pointer or be represented in storage (i.e. the value for the operand is to be retrieved from storage).
