Source/Sink Resource Not Available

The generic return code and device specific return code provide definition of the cause of this exception. The generic return code provides a generic reason code which is common to all source/sink objects of that type, and the device specific return code provides further device specific reasons for the generic return code.

The object of current user is only returned when the resource is in use by a source/sink object.

The number of devices in library and device record array data is only returned when a vary on for a tape library is performed or when the device allocation is changed. The list contains the attached resources and status of each resource.

The generic return code values for EX3404 are encoded as: Hex jknn, where:

j =
1 = Indicates a create instruction
2 = Indicates a modify instruction
4 = Indicates a materialize instruction

k =
1 = Indicates a ND object
2 = Indicates a CD object
3 = Indicates a LUD object
4 = Indicates a Request I/O instruction
7 = Indicates a NWID object
B = Indicates a AUXSV object

nn =

indicates the generic error code which provides further definition of the cause of the exception as follows:

For Create:
01 = Hardware not installed
02 = Support component not installed
03 = MFIOP expansion required

For Modify:
00 = Non-status element failure
01 = Set diagnostic mode or vary on (NWID,AUXSV) failure
02 = Vary on or continue (NWID) failure
03 = Answer (ND), activate (LUD), or cancel (NWID) failure
04 = Manual start data (ND) or dial out (CD) failure
05 = Enable failure
06 = Dial out failure
08 = Continue failure
09 = Cancel failure
11 = Resume failure
12 = Suspend failure
13 = Quesce failure

For Materialize:
11 = Logical channel status failed
12 = Logical link status failed
