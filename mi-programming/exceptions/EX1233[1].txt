</B></TD><TD><P><I>Data Space Index with Non-User Exit Selection Routine Build Termination
</I>
<P>While creating or rebuilding a data space index that contains a
non-user exit selection routine, data space entries were encountered
which resulted in an error in the selection routine.
The build, before terminating, found up to 20 instances of these types
of errors.
The instruction is terminated.
