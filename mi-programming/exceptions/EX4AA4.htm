Java Invalid Native Method Return

A Java native method has detached the current thread from the Java Virtual Machine (JVM) or destroyed the JVM and control is returned to an invocation executing a JNI or ATCHJVTH instruction that is newer than the oldest invocation executing a JNI or ATCHJVTH instruction.

If the thread issuing the DESJVM or DTCHJVTH instruction was initiated independently of the JVM, this exception is caused by the following condition:

  • An invocation exists that is currently executing a JNI instruction and control is returned to an invocation executing a JNI instruction that is newer than the oldest invocation executing a JNI instruction.
If the thread issuing the DESJVM or DTCHJVTH instruction was initiated by the JVM, this exception is caused by the following condition:

  • An invocation exists that is currently executing an ATCHJVTH instruction and control is returned to an invocation executing an ATCHJVTH instruction that is newer than the oldest invocation executing an ATCHJVTH instruction
