0607

Unsupported Space Use

This exception indicates one of the following conditions:

  1. An attempt has been made to use a teraspace storage address for an operation which does not allow a teraspace storage address. For example, a teraspace storage address may have been used as a parameter for a call to a program which is not teraspace capable.

  2. An attempt has been made to lock a teraspace storage address which is not mapped or allocated.

Instructions Causing Exception:

  • Call External (CALLX)
  • Cipher (CIPHER)
  • Lock Teraspace Storage Location (LOCKTSL)
  • Materialize Bound Program (MATBPGM)
  • Materialize Program (MATPG)
  • Scan With Control (SCANWC)
  • Scan Extended (SCANX)
  • Set Data Pointer (SETDP)
  • Set Data Pointer Addressability (SETDPADR)
  • Store Space Pointer Offset (STSPPO.htm)
  • Subtrace Space Pointer For Offset (SUBSPPFO)
  • Transfer Control (XCTL)
