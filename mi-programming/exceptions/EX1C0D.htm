1C0D

Requested Data Collection Function Not Valid

The function requested by an instruction may not be performed because of the current status of the machine, process, or thread.

Information Passed:
Offset
Dec Hex
Field Name
Data Type and Length
0 0
Reason code
Bin(2)
2 2
--- End ---

Below is the list of the reason codes and their meanings:
1 A request was received to start the collection of a category of data, while data collection for that category had already been started
2 A request was received to stop the collection of a category of data, while data collection for that category was currently stopped
3 A request was received to stop the collection of a category of data, from a process which was not the same process that started the collection
4 A request was received to start the collection of data into a byte stream file object which was already being used to collect data of a different category
5 A request was received to start the collection of data into a byte stream file which was not created to allow mapping into teraspace
6 A request was received to start the collection of data into a byte stream file object, whose data size was not less than the maximum data size specified with the request
