2403

Pointer Addressing Invalid Object Type

An instruction has referenced a system pointer that addresses an incorrect type of system object for this operation.

An instruction has referenced a space pointer that is supposed to point to an object extension but the space pointer does not point to an object extension.

Information Passed:
Offset
Dec Hex
Field Name
Data Type and Length
0 0 The invalid system pointer System pointer or Space pointer
16 10
--- End ---

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • Any instruction that uses a system pointer and requires a specific object type as a part of its operation
