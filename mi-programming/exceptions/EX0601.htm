0601

Space Addressing Violation

The exception indicates one of the following two conditions

  • an attempt has been made to operate outside the current extent of a space other than a teraspace.

  • an attempt has been made to reference an unallocated page within teraspace.

Note: Below, IPS stands for "implicit process space".

Different information is supplied for different space classes. Fields which are not used for a given space class will contain zero values. Here is a summary of the information returned (i.e. the field values set) for each space class.

    Hex 00 (primary associated space, including space objects)

  • system pointer to the object
  • offset
  • space class

    Hex 01 (secondary associated space)

  • system pointer to the object
  • offset
  • space class
  • secondary space number

    Hex 02 (auto)

  • system pointer to the Process Control Space
  • offset
  • space class
  • activation group mark
  • address

    Hex 03 (static)

  • system pointer to the Process Control Space
  • offset
  • space class
  • activation group mark
  • address

    Hex 04 (activation group-based heap storage)

  • system pointer to the Process Control Space
  • offset
  • space class
  • activation group mark
  • activation group-based heap space identifier
  • address

    Hex 05 (constant space)

  • system pointer to the object
  • offset
  • space class

    Hex 06 (handle-based heap storage)

  • system pointer to the Process Control Space that created the handle-based heap space
  • offset
  • space class
  • handle to handle-based heap space
  • address

    Hex 07 (teraspace)

  • system pointer to the Process Control Space
  • space class
  • flags
  • address
  • teraspace offset

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • Any non-bound program instruction that uses a space pointer to provide addressability to data used by the instruction. This includes operand basing pointers, space pointers used as operands that provide addressability, and space pointers in templates pointed to by an operand.

  • Any bound program instruction that uses a space address to provide addressability to data used by the instruction. This includes space addresses used as operands that provide addressability and space pointers in templates pointed to by an operand.
