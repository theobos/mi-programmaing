<TR VALIGN="TOP"><TD><P><B>3020
</B></TD><TD><P><I>Remote Journal Operation Error
</I>
<P>
The remote journal operation could not be performed.  The condition
described in the exception data identifies the reason for the failure.
<P>
This exception can be signaled for source and remote journal ports.
The <B><U>remote journal ID</U></B> will be binary 0 if signaled for a
remote journal port.
