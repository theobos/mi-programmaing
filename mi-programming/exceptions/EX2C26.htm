2C26

Invalid Activation Group

An invalid use of the shared activation group attribute was detected during program activation. If the client activation group is a shared type (activation group type of named shared or unnamed shared) then the service program activation group type must be one of the following:

  • callers
  • named shared with the same activation group name as the client activation group
  • unnamed shared
If the client activation group is not a shared type then the service program must not specify a shared type for its activation group.
