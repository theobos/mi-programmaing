4502

Invalid Request

Note: The term "heap space" in this exception description refers to either an "activation group-based heap space" or a "handle-based heap space".

The requested heap space operation is invalid.

The execution of the instruction is suppressed when the exception is signalled.

The error type codes for the invalid request are as follows:
Hex 0001 = Attempt to destroy the default heap space.

Note: Only applies to activation group-based heap spaces.

Hex 0002 = Attempt to free or reallocate heap space storage that is not allocated.
Hex 0003 = Access to the heap space not allowed.
Hex 0004 = Attempt to mark a heap space that cannot be marked.

Note: Only applies to activation group-based heap spaces.

Instructions Causing Exception:

  • Destroy Activation Group-Based Heap Space (DESHS)
  • Free Activation Group-Based Heap Space Storage (FREHSS)
  • Free Activation Group-Based Heap Space Storage from Mark (FREHSSMK)
  • Materialize Activation Group-Based Heap Space Attributes (MATHSAT)
  • Reallocate Activation Group-Based Heap Space Storage (REALCHSS)
  • Set Activation Group-Based Heap Space Storage Mark (SETHSSMK)
