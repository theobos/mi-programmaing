1C06

Machine Lock Limit Exceeded

The maximum number of concurrently held locks, implicit locks, and internal locks was exceeded.

Instructions Causing Exception:

  • Lock Object (LOCK)
  • Lock Object Location (LOCKOL)
  • Lock Space Location (LOCKSL)
  • Lock Teraspace Storage Location (LOCKTSL)
  • Unlink Teraspace Storage Location (UNLCKTSL)
  • Unlock Object Location (UNLOCKOL)
  • Unlock Space Location (UNLOCKSL)
