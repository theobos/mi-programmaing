1205

Data Space Entry Already Locked

An attempt has been made to lock a data space entry when the data space entry is already locked to a cursor (this cursor or another cursor) or to a commit block that is not attached to this process.

An attempt was made to alter a data space entry that was previous locked within the same secondary commit cycle. A system pointer to the process control space of the process that activated the cursor or attached the commit block that holds the lock is returned.
