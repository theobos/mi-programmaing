4004

Invalid Dump Data Retrieval

An attempt was made to retrieve dump data from a dump space but the retrieval was invalid.

Error codes and their meanings are as follows:

Code

Meaning

0001

The first block of dump data specified for retrieval is beyond the last block of dump data currently stored in the dump space.

0002

The data in the dump space is incomplete. The requirement is that all the data must be retrieved from a source dump space and inserted into the target dump space before the target dump space can be valid for the retrieval of data.
