2207

Authority Verification Terminated Due to Destroyed Object

Authority verification processing terminated due to a destroyed user profile or authority list found during the check of authority for a permanent system object.

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • Any instruction that references a permanent object through a system pointer.
