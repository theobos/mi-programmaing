The error class codes for the type of damage detected are as follows:
Hex 0000 = Unspecified abnormal condition
Hex 0002 = Logically invalid device sector
Hex 0003 = Device failure

The auxiliary storage device indicator field is defined for error classes hex 0002 and hex 0003. It is the OU number of the failing device or 0 for a main storage failure.

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.
