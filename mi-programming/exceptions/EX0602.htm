0602

Boundary Alignment

A program object has been referenced, and it does not have the proper alignment relative to the beginning of a space. Pointers must always be 16-byte aligned. Program objects that are not pointers must have at least the alignment specified by the ODT entry.

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • Any non-bound program instruction that has a pointer operand, a pointer providing addressability to an operand, or an operand that requires a specific boundary alignment
