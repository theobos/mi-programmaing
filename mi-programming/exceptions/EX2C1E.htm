2C1E

Activation Access Violation

The target program could not be activated within an existing activation group. The target activation group mark identifies the activation group. Either the effective program state or the effective storage model of the program is incompatible with that of the target activation group. The effective program state is the state required of the host activation group in order to activate and run the program. The activation group must have the same state. The effective storage model is the storage model required of the host activation group in order to activate and run the program. The requesting program is the program which is requesting the target program to be activated. The requesting program is null if the activation is being initiated by the machine.
