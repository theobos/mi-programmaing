0C22

Unpaired Shift Control

The source mixed data has an unpaired shift out control character (SO = hex 0E) and shift in control character (SI= hex 0F). If this exception is signalled for the XLATEMB instruction, the receiver field contains the data converted up to the point that the error was encountered. The length of the converted data is stored in the receiver converted data length field in the XLATEMB translation template.
