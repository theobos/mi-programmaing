1253

Invalid Journal for Referential Constraint Enforcement

An attempt has been made to change a data space that is not journaled to the single journal that is protecting the operations within the currently active secondary commit cyle. All operations executed within a secondary cyle that require referential integrity enforcement must be journaled to the same journal.

The data space field identifies the data space that the operation is occuring on.

The data space journal field identifies the journal that data space is journalled to.

The referential integrity network data space field identifies the data space that is associated with the RI network journal.

The referential integrity network journal field identifies the journal that is protecting the operations within the currently active secondary commit cycle.
