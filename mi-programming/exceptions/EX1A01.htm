1A01

Invalid Lock State

The lock enforcement rule or rules were violated when an attempt was made to access an object.

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • All instructions that enforce the lock rules. Refer to the Lock Enforcement section of individual instructions.
