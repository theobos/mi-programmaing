4C01

Asynchronous Signal Terminated MI Wait

A signal was received by a thread while in MI wait that caused the MI wait to be terminated. The exception is detected after the MI wait is entered.

The execution of the instruction is suppressed when the exception is signaled.

Information Passed:

  • None

Instructions Causing Exception:

  • Dequeue (DEQ)
  • Dequeue Wait (DEQWAIT)
  • Lock Object (LOCK)
  • Lock Object Location (LOCKOL)
  • Lock Space Location (LOCKSL)
  • Lock Teraspace Storage Location (LOCKTSL)
  • Wait On Time (WAITTIME)
