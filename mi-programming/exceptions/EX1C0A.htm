1C0A

Service Processor Unable to Process Request

An attempt was made to change the state of a function the service processor supports. The service processor was unable to process the request.

Meaning

00AC

A PC communications error has occurred or a request was made to change the key mode and either

  1. the system has a key slot but the key is not inserted, or

  2. the requested key mode is not available on this system model.

00AD

Invalid SRC value.

00AF

Time-of-day not available. A failure with the time-of-day circuitry has occurred, the time-of-day chip has failed.

The key mode is currently set to secure.
