2E01

User Profile Storage Limit Exceeded

The user profile specified insufficient auxiliary storage to create or extend a permanent object.

Information Passed:
Offset
Dec Hex
Field Name
Data Type and Length
0 0
User profile
System pointer
16 10
--- End ---

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • All create instructions creating a permanent object

  • All instructions extending a permanent object

    Any instruction which references bytes of data within a permanent space object can cause an automatic extension of the space if the space has the attribute of being automatically extendible. Therefore, this exception may be signaled for any instruction which has an operand which references bytes of data in a permanent space.
