3403

Source/Sink Object State Invalid

The exception indicates one of the following conditions:

  1. The source/sink object was not in the proper state or proper mode to allow execution of the instruction to complete successfully.

  2. An independent ASP number was specified for the instruction and the independent ASP is not varied on.

  3. A pointer to an independent ASP LUD was specified for the instruction and the independent ASP is not varied on.
