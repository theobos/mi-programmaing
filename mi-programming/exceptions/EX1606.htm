1606

Invalid Control Action Specified

A bound program exception monitor has specified an invalid control action or combination of control action and handler types.

Information Passed:
Offset
Dec Hex
Field Name
Data Type and Length
0 0
Reason code
Bin(2)



Hex 0000

Control action specified is not defined.

Hex 0001

The control action specified is Look At and the handler type is Branch point. Branch point implies "handling the exception", Look At implies "may/may not handle the exception". This combination is invalid.

Hex 0002

The control action specified is Handle and the handler defined is a default action which implies do not handle. This combination is invalid.
