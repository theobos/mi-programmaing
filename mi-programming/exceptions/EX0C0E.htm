0C0E

Floating-Point Zero Divide

This exception is signaled for a floating-point division operation if the divisor is zero and the dividend is a finite nonzero number.

Instructions Causing Exception:

  • Arc Cosine (ACOS)
  • Arc Sine (ASIN)
  • Arc Tangent (ATAN)
  • Arc Tangent Hyperbolic (ATANH)
  • Compute Math Function Using Two Input Values (CMF2)
  • Cosine (COS)
  • Cosine Hyperbolic (COSH)
  • Cotangent (COT)
  • Divide (DIV)
  • Exponential Function of E (EEXP)
  • Logarithm Base E (LN)
  • X to the Y Power (POWER)
  • Sine (SIN)
  • Sine Hyperbolic (SINH)
  • Tangent (TAN)
  • Tangent Hyperbolic (TANH)
