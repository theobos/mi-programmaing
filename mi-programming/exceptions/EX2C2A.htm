2C2A

Caller Parameter Mask Does Not Match Imported Procedure Parameter Mask

The parameter mask for a procedure imported from a service program does not match the parameter mask of the importer of the procedure. One cause of a mismatch is a difference in the teraspace capability of the calling and target procedures.
