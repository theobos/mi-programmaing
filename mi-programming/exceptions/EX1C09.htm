1C09

Auxiliary Storage Pool Number Invalid

An attempt was made to reference an auxiliary storage pool and the reference was not successful for one of the following reasons:

  • The auxiliary storage pool does not exist on the machine.

  • The auxiliary storage pool identifies an independent ASP and there is not an existing active logical unit description for the independent ASP
