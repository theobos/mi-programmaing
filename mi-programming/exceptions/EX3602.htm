3602

Invalid Space Modification

An attempt was made to modify the attributes of a space but the requested modification was invalid.

Error code hex values and their meanings are as follows:

Code

Meaning

0001

An attempt was made to modify the performance class attribute of the system object containing the space and the space was not a fixed length of size zero.

0002

An attempt was made to modify a system object to or from the state of having a fixed length space of size zero and the operation is invalid for that type of system object.

0003

An attempt was made to modify a system object to the state of having a fixed length space and the automatic extend attribute. These are mutually exclusive.

0004

An attempt was made to modify the associated space of a program while running in user state on a system running at security level 40 or above. This combination is invalid.

0005

The number of the secondary associated space to be created or modified is larger than the maximum allowed for this object.

0006

An attempt was made to modify the hardware storage protection enforcement of an associated space while running in user state.

0007

An attempt was made to modify the hardware storage protection enforcement of an associated space in a manner which would conflict with the hardware storage protection enforcement of the encapsulated part of the object.

0008

An attempt was made to create a secondary associated space for an object which was not created to allow secondaries.

0009

The number of the secondary associated space to be created already exists.

000A

An attempt was made to modify a packed secondary associated space.

000B

An attempt was made to create a secondary associated space while running in user state.

000C

An attempt was made to modify the hardware storage protection on a space of size zero, the space alignment was not chosen by the machine, the space of an object type that is not allowed to have this attribute changed, or the space does not exist.

000D

An attempt was made while running in user state to delete or truncate an associated space of an object which has secondary associated spaces.

0010

An attempt was made to modify the hardware storage protection enforcement of an associated space for an object type that is not allowed to have this attribute changed.

0011

An attempt was made to modify the expanded transfer size attribute of an associated space for an object type that is not allowed to have this attribute changed.
