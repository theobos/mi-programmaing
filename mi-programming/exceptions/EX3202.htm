3202

Scalar Attributes Invalid

A scalar operand did not have the following attributes required by the instruction:

  • Length

  • Precision

  • Boundary

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.

Instructions Causing Exception:

  • Any instruction using a late-bound (data pointer) scalar operand

  • Any instruction that verifies the length of a character scalar
