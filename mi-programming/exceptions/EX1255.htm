1255

Operation Conflict with Data Space Constraints

A data base operation was attempted on an object associated with data space constraint definitions. The requested operation conflicted with constraint definitions. The attempted operation could not be processed because one of the following conditions existed:

  • An operation was attempted on a data space that is associated with constraint definitions.

  • An operation was attempted on a data space index that is referenced by constraint definitions.
