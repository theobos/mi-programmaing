1604

Retry/Resume Invalid

An attempt was made to either retry a failed instruction or resume at an instruction after the instruction that failed, in an invocation where retry or resume is not allowed.

Alternatively, an attempt was made to execute an instruction that was not viewed by the machine as being reachable by the control flow in the program. This situation will rarely, if ever, occur.

Only the RTNEXCP instruction will actually receive this exception in non-bound programs, even though the condition was caused by some other instruction. In bound programs, the other individual instructions listed below may also receive this exception.

Instructions Causing Exception:

  • Modify Automatic Storage Allocation (MODASA)
  • Modify Automatic Storage Allocation (MODASA2)
  • Return From Exception (RTNEXCP)
  • Set Space Pointer From Pointer (SETSPPFP)
  • Any executed instruction viewed as unreachable by the machine
