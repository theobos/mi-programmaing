1C05

System Address Range Limit Exceeded.

The system's address range limit has been exceeded for a particular address size and type. An object has requested a new address to be added to the current object's size and that request has failed due to the system limit for that particular address range.

Return object pointer is the address of the object which the storage was being requested on behalf of. A value of binary zero is returned when the object address is unknown, in such conditions as a create of that object.

Auxiliary storage pool number indicates the auxiliary storage pool number in which the allocation was attempted.

Address size indicates the size of address which was attempted to be created.

Address type indicates the type of the address, permanent or temporary, which was attempted to be created.

Note: See Exception Specifications for a list of points to keep in mind when reading the descriptions of which instructions cause this exception.
