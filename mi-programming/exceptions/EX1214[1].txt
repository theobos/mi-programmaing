<TR VALIGN="TOP"><TD><P><B>1214
</B></TD><TD><P><I>Invalid Selection Template
</I>
<P>An error was detected in a selection template.
The <I>data space number</I> indicates which template in the selection
template list contains the error.
This field will equal 1 for a group-by selection template, and will
reference the appropriate template for the per data space mapping
template.
<P>The <I>offset to field in error</I> indicates which field is
in error in the selection template.
The offset equal 0 designates the field length of selection template is
in error, offset equal 4 designates the field number of selection
descriptors is in error, and so forth.
<P>The <B><U>invalid selection template token</U></B> field is the
information from the same-named field in the selection template header.
It is passed back unchanged by the machine.
<P>The invalid selection template exception will not be signaled when a
data pointer fails verification.
The normal exception associated with verifying data pointers will be
signaled instead.
